Setup:
1. Enable Developer Options (tap Build Number 7×)
2. Enable Wireless Debugging (Android 11+) or USB Debugging
3. For WiFi — go to Wireless Debugging → IP address and port and enter exactly what it shows (e.g. 192.168.100.7:38175)
4. Approve the pairing prompt on the device screen after clicking Connect
Find this on your device: Settings → Developer Options → Wireless Debugging → IP address and port
⚠ Android 11+ uses a random port — copy the full IP:PORT shown there, not just the IP.
